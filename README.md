####TCP&UDP

TCP与UDP均是运输层的运输协议，TCP面向连接，传输数据前要建立连接，UDP无连接，另外一个地方的主机收到UDP报文之后，不需要给出任何确认。

####ip协议&ip

ip通常指ip地址，是一个给因特网上的每一个主机或者路由器的每一个接口分配一个在全球范围内唯一的32位的标识符。ip协议是值TCP/IP体系中的一组协议，用于使互联的计算机网络进行通信。
